# JavaFX Memory Game

This project is a simple Memory (Pairs) game implemented in Java using JavaFX.

## Description
The application provides a graphical user interface where players flip cards and try to find matching pairs. The game keeps track of the current player and detects when all pairs are found.

This project was developed as part of a university programming assignment.

## Technologies
- Java
- JavaFX
- Maven
- Scene Builder

## Features
- Graphical user interface built with JavaFX
- Card matching game logic
- Player turn handling
- Winner detection

## Project Structure

src/main/java        - application source code  
src/main/resources   - UI files and images  
src/test/java        - unit tests  
pom.xml              - Maven configuration
